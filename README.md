# deploy-Branch — fertig gebaute Website (nicht bearbeiten)

Enthält nur die kompilierte, statische Website im Wurzelverzeichnis
(`index.html`, `assets/`, `.htaccess` …), für direktes Static-Hosting auf
Hostinger **ohne Build-Schritt**.

Quellcode/Entwicklung: Branch `main`.
Aktualisieren: auf `main` `npm run build`, dann Inhalt von `dist/` hierher kopieren.
